# Logos Directory

Place your affiliation logos in this directory with the following filenames:

- `shpe-logo.png` - SHPE logo
- `headstart-logo.png` - HeadStart logo
- `codepath-logo.png` - CodePath logo
- `colorstack-logo.png` - ColorStack logo
- `mlt-logo.png` - MLT logo

## Logo Guidelines

- **Format**: PNG format recommended (supports transparency)
- **Size**: Recommended max dimensions: 240px width Ã— 120px height
- **Aspect Ratio**: Logos will be automatically scaled to fit while maintaining aspect ratio
- **Background**: Transparent backgrounds work best
- **Fallback**: If a logo file is missing, it will be hidden automatically

The logos will display in grayscale by default and become colored on hover for a subtle, professional effect.
